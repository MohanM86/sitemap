Google Search Console

Slik bruker du Google Search Console med sitemaps.

Hjem/Google Search Console

Slik sender du inn sitemap

Gå til Search Console → Sitemaps → Skriv inn URL → Trykk «Send inn». Google vil crawle og rapportere status.

Hva du kan se

  • Dekningsrapport: Hvilke sider som er indeksert
  • Feilrapport: 404, redirect-feil, blokkerte sider
  • Sist lest: Når Google sist crawlet sitemappen
  • Oppdagede URLer: Antall URLer Google fant
Gratis
Search Console er 100% gratis og er det viktigste verktøyet for alle som jobber med SEO. Krever verifisering av nettstedseierskap.
